# Next.js Enterprise Starter

A production-ready Next.js template with TypeScript, Tailwind CSS, and best practices.

## ✨ Features

- ⚡ **Next.js 14** with App Router
- 🏆 **TypeScript** for type safety
- 🎨 **Tailwind CSS** for styling
- 🔐 **NextAuth.js** for authentication
- 💾 **Prisma** ORM with PostgreSQL
- 📊 **Tremor** for dashboards
- 🛠️ **ESLint + Prettier** configured
- 📱 **PWA** ready
- 🔧 **Docker** support

## 🚀 Quick Start

```bash
# Create from template
npx create-next-app@latest my-app --example https://github.com/jonathan7591/nextjs-enterprise-starter

# Or clone manually
git clone https://github.com/jonathan7591/nextjs-enterprise-starter.git
cd nextjs-enterprise-starter
npm install
npm run dev
```

## 📁 Project Structure
